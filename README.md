# TGT We Can – NGO Website

## Overview

TGT We Can is a website developed for an NGO to establish an online presence, share information about its initiatives, and provide a platform for community engagement.

The project focuses on accessibility, responsive design, and effective communication of the organization's mission and activities.

## Features

* Responsive website design
* Information about NGO initiatives
* Volunteer and participation information
* Contact and communication pages
* User-friendly navigation
* Mobile-friendly interface

## Objectives

* Increase awareness of NGO activities
* Improve accessibility of information
* Create a professional digital presence
* Enable easier communication with supporters and volunteers

## Tech Stack

* HTML
* CSS
* JavaScript
